# 日报 / 周报 / 月报填写规范

本章节用于统一 **日报、周报、月报的填写目标与模板**，用于团队协作、进度同步与复盘沉淀。  
⚠️ **报表格式如有不确定，请优先询问：粟哥**。

---

## 日报填写

### 日报目标
- 记录 **当天工作与交付**
- 更新 **当前问题与风险**
- 明确 **明日行动计划**
- 同步并维护 **周计划列表**

---

### 日报填写说明（重点）
- 日报需 **整体复制模板表格**
- 日报中 **必须包含周计划列表**
- 周计划列表需 **每日更新进度状态**

日报模板参考链接：  
https://docs.google.com/spreadsheets/d/1_wiJrdGU_7QXrYVOuzllHu_4K48lF-C5f6uWn-ojiOU/edit?gid=1281751265#gid=1281751265

---

### 日报模板（内容项）

| 模块 | 填写内容 |
|----|----|
| 今日感想 | 今日整体感受与状态 |
| 今日问题 | 背景、问题、思考 |
| 今日工作 | 今日完成的具体事项 |
| 明日计划 | 明日需要完成的工作 |
| 周计划列表 | 本周整体计划与当前进度 |

---

## 周报填写

### 周报目标
- 总结 **一周整体感想**
- 复盘 **本周得失**
- 梳理 **本周工作产出**
- 明确 **下周计划**

### 周报模板（内容项）

| 模块 | 填写内容 |
