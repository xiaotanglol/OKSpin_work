# 名词解释

## 技术相关术语

-   **PR / MR**：Pull Request / Merge Request，用于代码合并与审核流程。
-   **CDN**：内容分发网络（Content Delivery Network），用于加速静态资源加载。
-   **埋点 / 数据埋点**：用于记录用户行为并进行数据上报的技术手段（如点击、曝光、转化等）。
-   **首屏**：用户打开页面后首次可视内容完成加载所需的时间或资源状态。

## 投放与业务相关术语

-   **直媒**：直接媒体投放模式，通常为固定跳转或简化流程版本。
-   **RTB**：Real-Time Bidding，实时竞价广告投放模式，根据流量与用户实时竞价展示内容。
-   **SG 定制**：针对特定平台或渠道（如 SG）进行的定制版本开发，通常包含特殊逻辑或参数。
-   **定制跳转 EC**：根据投放场景或用户行为进行定制化跳转的 End Card 页面逻辑。

## 基础数据指标

-   **UV**：Unique Visitor，独立访客数（去重后的访问用户数量）。
-   **PV**：Page View，页面浏览量（页面被访问的总次数）。

## 玩法数据指标

-   **Tmlview**：完整进入玩法页面的次数。
-   **Tmlview（user）**：完整进入玩法页面的用户人数（去重）。

## 互动相关指标

-   **play**：用户进入页面后产生互动行为的次数（例如点击按钮、抽奖等行为）。
-   **play（user）**：产生互动行为的用户人数（去重）。

## 互动埋点说明

-   **showReward**：互动奖励展示次数（互动展现）。

-   **getReward**：用户成功获得奖励的次数，通常用于计算互动率。

-   **clickReward**：用户点击奖励素材的次数（互动点击）。

## 广告素材指标

-   **Impr**：中奖素材或广告素材的展示次数。  
    一般情况下：
    ```
    Impr ≤ play
    ```
    即：一次互动通常只会展示一个素材。
-   **Click**：用户点击素材的次数，该数据通常与素材吸引力和用户兴趣相关。

## 展现率指标

-   **Playimprrate**：互动后素材展示率。
    通常在：
    ```
    90% - 100%
    ```
    说明：
-   一次互动最多展示一个素材
-   若广告预算不足、素材填充失败或系统异常，可能导致该指标降低
-   如果明显低于 90%，通常需要排查异常情况

## EC 页面指标

-   **ECView**：用户点击中奖素材后进入 EC 页面并成功展示的次数。

## 转化指标

-   **CVR**：转化率（Conversion Rate）。
    计算公式：
    ```
    CVR = Install / Click
    ```
    用于衡量点击后最终完成安装或转化的比例。

## 数据流程关系（简化理解）

```
Tmlview
↓
play
↓
Impr
↓
Click
↓
ECView
↓
Install
```

说明：

-   用户进入玩法页面 → 产生互动
-   互动后展示素材 → 用户点击素材
-   点击素材进入 EC 页面 → 最终产生转化（Install）
