## 推荐阅读顺序（新人）
1. 入职引导（第一天）：账号/权限/平台、团队与技术栈、开发环境与工具、第一天任务
2. 工作流程：需求 → 开发 → 提测 → 上线  
3. 项目交接：目录结构、运行构建、发布、风险点  
4. 技术规范：代码规范、Vue 约定、资源与性能  
5. AI 与效率工具：工作流、Prompt、工具清单  
6. 日报/周报/月报填写

## 如果你卡住了
- 先用右上角目录（或左侧目录）定位
- 仍不清楚：把“你做了什么、看到什么、期望什么”写清楚再问同事
