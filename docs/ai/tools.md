# 工具清单

> 本清单为推荐工具集合，团队不强制统一编辑器与大部分辅助工具，成员可根据个人习惯选择。

---

## 编辑器 / IDE

-   编辑器不做统一要求，可使用市面常规主流编辑器  
    （如 VS Code、WebStorm、Sublime Text 等），重点保证：

    -   能稳定运行项目
    -   支持 Git 工作流
    -   支持插件扩展
    -   支持前端调试与终端集成

-   AI IDE 可按个人习惯选择  
    （如 Cursor 或其他 AI IDE），但需遵守团队 AI 使用规范：

    -   AI 生成代码必须人工 Review
    -   禁止直接提交未经验证的 AI 代码
    -   重要逻辑需补充注释说明

-   允许使用常见 AI 开发插件或工具辅助编码，例如：

    -   代码补全与生成工具
    -   文档生成与注释工具
    -   重构与调试辅助工具

    使用原则：

    -   不影响项目可读性
    -   不破坏团队编码规范
    -   不引入安全风险或隐私泄露

---

## 辅助工具

-   Git 可视化管理：TortoiseGit
-   图片压缩：TinyPNG
-   图片格式转换：FileConverter
-   音频编辑：GoldWave
-   骨骼动画编辑：Spine

> 辅助工具不做强制统一，按项目需要自行选择。

---

## 验收标准 ✅

新人应能够独立完成以下基本能力：

-   编写与运行项目代码
-   提交与管理 Git 版本
-   基本问题排查与定位
-   阅读并理解团队文档
