# 交接 Checklist（必对照）

## 代码
- [ ] 主分支可运行（拉下来即可跑）

- [ ] README/文档已更新（命令、入口、注意事项）

## 构建 & 发布
- [ ] 构建命令已验证（本机/CI）
- [ ] 发布路径已说明（平台/服务器目录）
- [ ] CDN/域名/缓存策略已记录
- [ ] 回滚方案明确

## 业务
- [ ] 核心玩法说明（主流程 + 边界）
- [ ] 关键状态说明（次数、中奖、登录）
- [ ] 已知问题/风险点记录（兼容/性能/三方）

## 资源
- [ ] 素材来源清楚（设计稿/导出）
- [ ] 动画资源（Spine/Lottie）版本与导出方式
- [ ] 大图压缩与格式（webp/png）规范
